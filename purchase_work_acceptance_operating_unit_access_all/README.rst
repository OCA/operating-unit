This module allow a user to have Access all OUs' work acceptances,
without having to add OUs in user setting.

Example use case, a shared procurement team under an OU
but need to work on work acceptances of all OUs.

   :local:

Configuration
=============


To configure a user to have access to all OUs' work acceptances (without all OUs access):

* Go to *Settings / Users & Companies / Users*
* For a user, select checkbox "Access all OUs' work acceptances"
